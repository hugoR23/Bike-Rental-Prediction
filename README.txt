Ce fichier decrit les élements clefs de cette archive:
# ./data
	Ce dossier regroupe l'ensemble des données bruts fournie par BayArea BikeShare ainsi que les données météo bruts.

# ./QGIS Project
	Ce dossier regroupe les fichier du projet QGIS permettant d'extraire des informations démographiques pour chaque station.

# ./SAS Project
	Ce dossier est composé du diagramme SAS Entreprise Miner permettant la prédiction ainsi que de l'ensemble des données transformées dans le dossier input.

# ./Script/convertRentalData.js
	Ce script permet de convertir les données bruts de location en données traffic utilisées pour la prédiction. Le script est en javascript et s'execute avec Node.Js.

# ./Script/getPlaceAroundStations.html
	Ce script permet de reunir des informations sur le nombre d'établissements autour des stations. Il s'execute simplement dans un navigateur avec une connexion internet
